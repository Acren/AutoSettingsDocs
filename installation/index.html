Installation

Auto Settings is available as a plugin through the Unreal Marketplace.

Installation is typical for an Unreal plugin:

  1. Install through the Epic Games Launcher
  2. Enable the plugin in the Edit > Plugins menu in your project

At this point the plugin should be enabled in your project and ready for you to use.

If you want to use the menu or assets from the example project, read how to migrate them here.

Or, you can move on to using the UMG Widgets to add settings and input bindings to your own menu.
